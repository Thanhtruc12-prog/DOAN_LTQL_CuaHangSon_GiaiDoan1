# LTQL_CuaHangSon
- Giai đoạn 1: Em tiến hành xác định các bảng (models) cần có trong cơ sở dữ liệu đề tài "Quản lý cửa hàng bán son".
- 13 bảng bao gồm các bảng: Tài khoản, Phân quyền, Vai trò, Nhân viên, Khách hàng, Sản phẩm, Loại sản phẩm, Hóa đơn, Chi tiết hóa đơn, Phiếu nhập, Chi tiết phiếu nhập, Tồn kho, Nhật ký hoạt động (Log Table).
- Đã code được các bảng: Sản phẩm, Nhân viên, Tài khoản, Phân quyền, Vai trò, Loại sản phẩm, Nhà cung cấp.
- Word tiến hành xác định tổng quan về đề tài
